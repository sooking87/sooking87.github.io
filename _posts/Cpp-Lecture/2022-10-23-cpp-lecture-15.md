## 최대 공약수 구하기

```cpp
// 최대 공약수
void postDiv(int &denum, int &num)
{
    int div = 0;
    int min = denum > num ? num : denum;

    for (int i = min; i > 0; i--)
    {
        if (denum % i == 0 && num % i == 0)
        {
            div = i;
            break;
        }
    }

    denum /= div;
    num /= div;
}
```

## 최소 공배수 구하기

```cpp
// 최소 공배수
long long getMul(int n1, int n2)
{
    long long max = n1 >= n2 ? n1 : n2;
    cout << "max: " << max << endl;
    while (true)
    {
        if (max % n1 == 0 && max % n2 == 0)
            return max;
        max++;
    }
}
```
