## 문제

**\*** <br> \***_ <br>
_** <br> \*_ <br>
_ <br>

이렇게 별찍는 문제

## 풀이

```cpp
#include <iostream>

int main()
{
    int n;
    std::cin >> n;

    for (int i = n - 1; i >= 0; i--)
    {
        for (int j = n - 1; j >= 0; j--)
        {
            if (i >= j)
            {
                std::cout << '*';
            }
            else
            {
                std::cout << ' ';
            }
        }
        std::cout << std::endl;
    }
}
```
