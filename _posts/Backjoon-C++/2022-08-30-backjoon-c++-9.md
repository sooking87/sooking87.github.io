## 문제

N이상 M 이하의 수 중에서 소수 구하기

## 틀림

```cpp
#include <iostream>
#include <cmath>

int main()
{
    int start;
    int end;
    int num;
    std::cin >> start >> end;

    for (int i = start; i <= end; i++)
    {
        num = sqrt(i);
        // 1이면 소수가 아니고 2, 3은 소수이다.
        if (num == 1 && i != 1)
        {
            std::cout << i << std::endl;
            continue;
        }
        else
        {
            int isPrimeNum = true;
            for (int j = 2; j <= num; j++)
            {
                if (i % j == 0)
                {
                    isPrimeNum = false;
                    break;
                }
            }
            if (isPrimeNum)
            {
                std::cout << i << std::endl;
            }
        }
    }
}
```

처음에는 입력받은 수까지 반복문을 돌리면서 루트값까지 값을 나누면서 나누어 떨어지는지 아닌지를 확인하면 될 줄 알았다. ..... js 도 그렇게 풀었기 때문ㅇ,,,, 근데 왜 틀림,,?!?!?!?
