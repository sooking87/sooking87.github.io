## 🔮 JSX 사용법

### 1️⃣ 사용자 지정 태그 만들기

1. src > 파일 생성

   ```javascript
   //src > MyHeader.js
   const MyHeader = () => {
     return <header>MyHeader</header>;
   };

   export default MyHeader;
   ```

2️. src > App.js

    ```javascript
    import "./App.css";
    import MyHeader from "./MyHeader"; //🌟 넣고 싶은 태그 파일 임포트 하기

    function App() {
        return (
            <div className="App">
            <MyHeader />
            <h2>안녕 리액트</h2>
            <b>
                React.js <br></br>
            </b>
            </div>
            //닫는 태그 반드시 필요! <img /> <br /> 태그도 반드시 닫는 태그 필요
        );
    }

    export default App;
    ```

3️. npm start를 통해서 결과 확인
